#  Essentials

Below you will find the essentials of what you need to quickly start using Stormy Linux shortcuts.

### Essentials

| Command                           | Description                        |
|-----------------------------------|------------------------------------|
| `mod` + `return`                  |Default Terminal(Kitty)             |
| `mod` + Ctrl + f                  |GUI File Explorer (Thunar)          |
| `mod` + f                         |Terminal File Explorer (Ranger)     |
| `mod` + Ctrl + k                  |Advanced File Explorer (Krusader)   |
