# Get started with Stormy Linux

## Start using SL

Getting started with Stormy Linux is a simple and easy process. After the release of the Alpha, Stormy Linux is now available to be run on a Debian based distribution. It is also possible to run the Stormy Linux configuration on any other Debian or Arch based distribution, this will require extra configuration and a basic knowledge of Linux, Python and window managers.

To get started using the Alpha of the Stormy Linux configuration, continue to the next page.
